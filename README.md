YouTube のコメント欄の位置を変える Chrome 拡張機能

## 概要

この拡張機能は，YouTube のコメント欄の表示位置を自由に変更できるツールです．  
通常，YouTube のコメント欄は動画のメタデータ（タイトル，再生回数，概要欄など）の下に表示されますが，本拡張機能を使用すると，コメント欄を動画プレイヤーの横や直下に移動できます．

「コメントを見ながら動画を楽しみたい」という方におすすめの拡張機能です．

## 特徴

- コメント欄を動画の横（サイドバー）に配置可能
- サイドバーだけでなく，動画の直下や動画メタデータの下にも配置可能
- コメントを見ながら快適に動画を視聴可能
- 画面サイズに応じて最適なレイアウトを自動で適用
- サイドバー幅をドラッグで調整でき，コメント欄をより広く表示可能

## 仕様

- ウィンドウ幅が 1017px 以上の場合，コメント欄を動画の横に配置できます．
- ウィンドウ幅が 1017px 未満の場合，コメント欄は動画の下，またはメタデータの下に配置されます．
