## 拡張機能の使い方

1. Chrome ウェブストアから拡張機能をインストールする
   - [YouTube コメントポジション - Chrome ウェブストア](https://chrome.google.com/webstore/detail/neclohaabpkkpcmckmanifdabgdnnmjd) にアクセスします．
   - 「Chrome に追加」ボタンをクリックして拡張機能をインストールします．
   - インストールが完了すると，ブラウザのツールバーに「YouTube コメントポジション」のアイコンが表示されます．

2. コメント欄の位置を設定する
   - ブラウザのツールバーにある拡張機能のアイコンをクリックし，ポップアップ画面を開きます．
   - Large レイアウト または Medium レイアウトごとに，コメント欄の設定を行います．
     - コメント欄の高さを設定できます．
       ※未入力の場合は，自動で最適な高さが設定されます．
     - コメント欄の配置を選択します（動画の横，動画の直下，メタデータの下 など）．

3. ツールを有効にする
   - 「YouTube コメントポジション を有効にする」のトグルボタンをクリックして，ツールを有効にします．
   - 無効にする場合も同じ操作を行ってください．

4. YouTube の動画ページで確認する
   - YouTube の動画ページを開きます．
   - 設定どおりにコメント欄が表示されていることを確認します．
     正しく適用されていれば完了！ 🎉

### 注意事項

- 設定を変更した後は，ページを再読み込みする必要があります．
